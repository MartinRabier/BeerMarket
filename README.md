🍺 Beer Market

Beer Market is a fun and interactive simulation of a stock market… but for beer 🍻
Prices evolve dynamically, influenced by randomness and user actions, creating a playful trading environment.

🚀 Concept

Beer Market is a real-time simulation where different types of beers behave like financial assets:

Prices fluctuate randomly over time 📈📉
Users can interact with the market
Events like sales, crashes, and bull runs directly impact prices
🎮 Features
📊 Dynamic Pricing
Each beer has its own price evolution
Prices follow a stochastic/random behavior
💰 Market Actions
Sell a beer → impacts its price (market reaction)
Simulates supply/demand dynamics
⚡ Market Events
Crash 💥
→ sudden drop in price
Bull Run 🚀
→ strong upward trend

These events allow you to manually influence the market and create volatility.

🍻 Available Assets

The simulation includes several types of beers:

Chouffe
Pils
Kriek
Stout

Each with its own price trajectory and behavior.

🧠 Purpose

This project is a personal and experimental project developed for:

🎉 Entertainment within a student association
🧪 Exploring basic financial market mechanics
📊 Practicing data visualization and simulation
🛠️ Tech Stack
Python
Data visualization (Matplotlib / similar)
Basic simulation logic (random processes)
📈 Example
Prices evolve over time
User actions (buy/sell/events) create visible impacts
Market becomes more dynamic and interactive
🎯 Future Improvements
Add a real trading system (buy/sell portfolio)
Introduce player scores or leaderboard
Web interface (React / Streamlit)
More complex price models (volatility, trends, AI)
🤝 Context

Developed as part of a student association project, Beer Market is designed to bring fun, interaction, and a bit of finance culture into a social environment.

⚠️ Disclaimer

This is not a financial tool — just beer, fun, and chaos 🍺📉
